# Sapy C++ Libraries

Sapy is a collection of C++ libraries that are as "simple as Python".


## Build 

**Default Build (Release):**

```sh
./build.sh  # Builds the project in Release mode
```

**Debug Build:**

```sh
./build.sh Debug  # Builds the project in Debug mode
```
**Generate Conan Package:**

```sh
./package.sh
```
